# Smart-Doc-AI-Agent
Kullanıcının yüklediği PDF veya metin dosyaları üzerinde hem doğal dilde sorulara cevap veren, hem de bu dosyalardaki verilerle işlem yapabilen (grafik çizme, özetleme, veri ayıklama) bir sistem
LLM'ler bağlam penceresi (context window) sınırlı olduğu için veriyi önceden işlemek gerekir. Bu modül, yüksek boyutlu dokümanları bellek dostu ve hızlı bir şekilde tokenize edilmeye hazır hale getirir.
